# Bonus Artwork Prompts - Tech-Fantasy Matrix/Tron

Objectif: generer un artwork carre (1:1) par bonus, lisible en petite taille, avec une vibe informatique futuriste (tech-fantasy, matrix, terminal, neon cyber).

## Global Style Guide (a reutiliser pour tous les prompts)

Prompt de base:
"Square game icon, centered subject, high readability at 128x128, bold silhouette, clean composition, tech-fantasy cyber theme, matrix code motifs, dark metallic background, neon accents (cyan + acid green + amber), subtle glitch artifacts, terminal UI micro-details, volumetric rim light, high contrast, stylized painterly pixel-hybrid look, no text, no logo, no watermark"

Negative prompt:
"photorealistic, cluttered composition, tiny unreadable details, flat lighting, washed colors, watermark, text, signature, blurry"

Parametres recommandes:
- Format: 1:1
- Cadrage: sujet principal central, macro iconique
- Lisibilite: 3 niveaux de lecture (forme globale, symbole secondaire, details)

## Conseils de coherence visuelle
- Garder le meme angle de camera entre toutes les icones (quasi frontal 3/4).
- Garder la meme epaisseur de contour du sujet principal.
- Associer une couleur dominante secondaire par archetype:
  - General/meta: vert + cyan
  - Firewall: ambre + orange
  - Mage: cyan + violet froid tres discret
  - Rogue: vert acide + blanc electrique
- Toujours privilegier une silhouette simple lisible a tres petite taille.
